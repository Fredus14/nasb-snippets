# Normalization
## Z-score Normalization


```r
original <- matrix(c(
    2,  4,  4,
    5,  4,  14,
    4,  6,  8,
    3,  5,  8,
    3,  3,  9
), ncol = 3, byrow = TRUE)
original
```

```
##      [,1] [,2] [,3]
## [1,]    2    4    4
## [2,]    5    4   14
## [3,]    4    6    8
## [4,]    3    5    8
## [5,]    3    3    9
```



```r
means <- rowMeans(original)
stddevs <- apply(original, 1, sd)
cbind(means, stddevs)
```


```r
centered <- original - means
centered
```
